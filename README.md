
🥬 FreshnessGuard AI — Intelligent Retail Optimization Dashboard

Author: Preethin Peter

Organization: Wise Analytics

Project Duration: February 9, 2026 – April 9, 2026

🚀 Project Overview

FreshnessGuard AI is an end-to-end data science solution designed to reduce retail food waste and optimize inventory decisions using machine learning, forecasting, and AI agents.

This project combines predictive analytics, time-series forecasting, and multi-agent decision systems into a unified dashboard that enables smarter retail operations.

🎯 Key Objectives


Predict product spoilage risk at store and category level
Forecast demand for the next 30 days
Recommend inventory and replenishment actions
Reduce overall waste and operational inefficiencies
Simulate AI-driven decision making using agent workflows


📊 Features

📌 1. Overview & Exploratory Data Analysis (EDA)
Waste distribution by category
Demand trends and seasonal patterns
Correlation heatmap for key variables
Monthly performance insights

🤖 2. Spoilage Prediction Model

Binary classification model to predict spoilage risk
Confusion matrix and ROC curve evaluation
Feature importance analysis
Store-level risk segmentation

📈 3. Demand Forecasting
30-day demand forecast using:
XGBoost (machine learning)
Prophet (time-series model)
Confidence intervals for predictions
Aggregated demand across stores

🧠 4. AI Agent System
Multi-agent workflow powered by LangChain
Inventory Agent
Demand Agent
Weather Agent
Decision Agent
Automated reasoning for operational decisions

♻️ 5. Waste Reduction Engine
Before vs after waste comparison
Recommended actions for optimization
Replenishment signals and alerts

🖥️ 6. Interactive Dashboard
Built using Streamlit with a dark professional theme
Fully interactive visualizations using Plotly
Modular sections for business storytelling

🛠️ Tech Stack
Component	Technology Used
Dashboard	Streamlit
Visualization	Plotly
Machine Learning	scikit-learn, XGBoost
Time Series	Prophet
AI Agents	LangChain + Ollama (LLaMA3)
Styling	Custom CSS (Dark Theme)
